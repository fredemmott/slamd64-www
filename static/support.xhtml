Support is available from several sources - please try in this order:

  1. The FAQ.
  2. The Slackware Book.
  3. The Slamd64 bugtracker if you think you've found a bug - please check to make sure it's not already there first.
  4. The Slamd64 forums - please check your question hasn't already been asked (and answered) by someone else.
  5. #slamd64 on Freenode IRC.
  6. As a last resort, you can contact me personally - this is usually the slowest of any of the options.
